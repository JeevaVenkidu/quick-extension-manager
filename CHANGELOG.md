# Changelog

All notable changes to Quick Extension Manager are documented here. This file follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project uses [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-18

### Added

- Manifest V3 Chrome extension popup for managing installed extensions.
- Alphabetical extension listing separated into Active and Inactive groups.
- Live name search and empty states for missing or unmatched extensions.
- Individual enable/disable toggles backed by the Chrome Management API.
- Confirmed bulk enable and disable actions.
- Accessible labels, focus states, icon fallbacks, loading state, and reduced-motion support.
- GitHub Actions validation and tagged-release packaging.
